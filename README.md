# üè¶ Janasiri Motors - Cheque Management System (Frontend)

This project is the **frontend application** of the Cheque Management System developed for **Janasiri Motors**.  
It provides an intuitive interface to manage, track, and monitor issued and received cheques with real-time updates.

---

##  Features

-  Manage issued and received cheques easily  
-  Track cheque due dates and payment status  
-  Dashboard with analytics and visual summaries  
-  Smart filtering and search for cheque records  
-  Integration-ready with backend APIs  
-  Responsive and user-friendly interface  

---
